# dotfiles

> You don't really know someone until you look at his dotfiles.

My dotfiles.

## vscode

### vscode dotfiles pull

Pull the vscode dotfiles from Windows:

```powershell
cd .\vscode\windows\
.\pull.ps1
```

### vscode dotfiles push

Push vscode dotfiles into Windows:

```powershell
cd .\vscode\windows\
.\push.ps1
```
